## Create a new file

To create a new `How-to` Article, create a new file under the `docs -> how-to` directory as shown below

    workshop # Root directory of your repo
    └── docs
       ├── how-to
            ├── new-howto-article.mdx

:::note -
[Refer this link](./sidebar#files) for file naming instructions.
:::

## Update the content

Update the file with relevant content. [Refer this link](../mdx/intro) for mdx intro

## Update _sidebar.json

**_sidebar.json** is located relative to the directory, in this instance, the file can be found at
`docs -> how-to -> _sidebar.json`

    workshop # Root directory of your repo
    └── docs
       ├── how-to
            ├── _sidebar.json


:::note -
[Refer this link](./sidebar#sidebar) for sidebar documentation.
:::
