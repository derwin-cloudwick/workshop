In addition to the basic Markdown syntax, we use remark-admonitions alongside MDX to add support for admonitions. Admonitions are wrapped by a set of 3 colons.

Examples:

## Note

    :::note
    this is a note from admin. It **can** also include markdown.
    :::

the above code block generates the following output.

:::note
this is a note from admin. It **can** also include markdown.
:::

## Tip

    :::tip
    Give a tip to the user on best practice.
    :::

the above code block generates the following output.

:::tip
Give a tip to the user on best practice.
:::

## Info

    :::info You can specify optional title here
    This is some useful information, Notice that this has title instead of info wording
    :::

the above code block generates the following output.

:::info You can specify optional title here
This is some useful information, Notice that this has title instead of info wording
:::

## Caution

    :::caution
    Caution user, there is something you need to pay more attention to
    :::

the above code block generates the following output.

:::caution
Caution user, there is something you need to pay more attention to
:::

## Danger

    :::danger
    Mayday! Mayday! Mayday!
    :::

the above code block generates the following output.

:::danger
Mayday! Mayday! Mayday!
:::
