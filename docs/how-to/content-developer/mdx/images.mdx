Regular Markdown images are supported.

Add an image at `static/img/logo.svg` and use this Markdown declaration:

```md
![Amorphic logo](/img/core/logo.svg)
```

![Amorphic logo](/img/core/logo.svg)

Or additionally you can add an image at `./media/logo.svg` and use this Markdown declaration with relative path:

```md
![Amorphic logo](./media/logo.svg)
```

![Amorphic logo](/img/core/logo.svg)
