## Relative paths
Regular Markdown links are supported using url paths or relative file paths.

```md
Let's learn more about MDX [here](./intro).
```

Let's learn more about MDX [here](./intro).

<h3>Or</h3>

```md
Let's go back to [Use cases List](../../../use-cases).
```

Let's go back to [Use cases List](../../../use-cases).


## HTML tags
you can also use HTML anchor tags to create links

```html
Visit <a href="https://amorphicdata.com">Amorphic Data</a> for more information
```
Visit <a href="https://amorphicdata.com">Amorphic Data</a> for more information
